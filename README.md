This is an iterative prewhitening tool for BlackGEM light curves. 

The following need to be manually specified before running the code. 

- Specify the light curve directory and filename as here: lc=fits.open("BG_lightcurves/lightcurves_4targets.fits")

- Then select the object ID. Here it is the Gaia DR3 ID: 

- Finally, select the filter in Filter

There are other parameters that might be useful to define, such as the frequency search range and number of iterations.

This code is still being improved so that it can be run without editing the code internally.
